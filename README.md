## 🌟 Introduction

**Founder's Link** is the ultimate platform for connecting visionary startup founders with enthusiastic investors. In today’s competitive startup ecosystem, the right connection can be a game-changer. Founder's Link acts as your digital bridge by:

- 💡 Allowing **founders** to showcase their groundbreaking ideas.
- 📈 Helping **investors** discover innovative startups that align with their goals.

Our mission? **Empowering innovation and fostering meaningful partnerships** to drive growth and creativity in the entrepreneurial world.

---


## 🔥 Core Features

### 🏠 **Landing Page**
- A sleek and interactive homepage that welcomes users, explaining the platform’s value and guiding them through a seamless onboarding experience.
- Dynamic visuals and animations built with **HTML**, **CSS**, and **Bootstrap** to engage users at first glance.

### 📤 **Pitch Submissions**
- **Founders** can create detailed startup pitches that include:
  - Comprehensive business models.
  - Market strategies and financial projections.
  - Attached multimedia files (documents, videos) using **Multer** for impactful presentations.
- Automated email notifications to relevant investors upon pitch submission via **Nodemailer**.
- Cloud storage support for secure and scalable file management.

### 🔍 **Search and Filter Options**
- **Investors** can effortlessly search and filter startups based on:
  - 🌍 Industry
  - 🏢 Funding Stage
  - 📍 Location
  - 📊 Keywords for niche searches.
- An intelligent recommendation engine (powered by **Node.js**) suggests startups matching an investor’s portfolio.

### 🤝 **Connection Requests**
- Personalized connection requests enable **investors** to express interest in specific startups.
- **Founders** are notified instantly, ensuring timely follow-ups.
- Status tracking for sent and received requests.

### 💬 **Secure Messaging System**
- Real-time private chat powered by **Socket.io** with:
  - 📝 Editable and deletable messages.
  - 📂 File sharing for pitch-related discussions.
  - 🔄 Instant syncing across devices for seamless communication.
- Advanced chat features like read receipts and typing indicators for an engaging user experience.

### 📊 **Deal Management Dashboard**
- Investors can organize their interactions with startups via:
  - Visual status tracking of ongoing deals.
  - Insights into investment progress with detailed analytics.
  - Scheduled tasks and reminders to keep deals on track.

### 🚀 **Funding Progress Tracker**
- **Founders** can monitor their funding rounds in real-time:
  - Track investor engagement and responses.
  - Visual graphs for funding milestones.
  - Customizable notifications to stay informed.

### 🏅 **Investor Ranking**
- A leaderboard highlights active and reputable investors:
  - Rank based on engagement metrics like number of deals and feedback.
  - Helps founders identify high-value connections.

### 🗓 **Meeting Scheduler**
- Built-in scheduling tools to arrange virtual meetings:
  - **WebRTC** and **Agora.io** for high-quality video conferencing.
  - Calendar integrations for synchronized scheduling.
  - Automated reminders to ensure no meetings are missed.

### 🔐 **Two-Factor Authentication (2FA)**
- Secure your account with **email-based OTP verification**.
- Adds a robust layer of protection for sensitive actions like profile updates or messaging.

## 🛠 Project Setup

### 📋 Prerequisites
Make sure you have the following installed:
- 🛠 **Node.js**
- 📦 **npm**

### 📥 Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Devdesai1901/Founder-sLink_grp12.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Founder-sLink_grp12
   ```
3. Seed the database with dummy data:
   ```bash
   cd server/tasks
   node seed.js
   ```
4. Navigate back to the root directory and install dependencies:
   ```bash
   cd ../..
   npm install
   ```
5. Start the development server:
   ```bash
   npm start
   ```
6. Open your browser and explore:
   ```
   http://localhost:3000
   ```
